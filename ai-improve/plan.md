# Improvement Plan

## Goals
Transform the AI News Page into a clean, robust, production-ready Streamlit application with good error handling, caching, UI polish, and maintainable code structure.

## Phases
1. Analyse — audit codebase and document issues
2. Plan — prioritise tasks, define standards
3. Refactor — code quality, structure, error handling, caching
4. UI — improve layout, card design, loading states
5. Test — add unit/integration tests
6. Polish — final documentation, README, requirements.txt
